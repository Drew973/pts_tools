Point to Chainage

Algorithm to find nearest section and chainage to points.Creates new layer:

  • With same fields as input plus a text field named section and a double field named chainage.
  • Containing all features of input.
  • With null section and chainage where no section found.

Parameters:

  • Input layer: layer with points
  • Layer or data source with network: Defaults to L:\SharedDocs\HAPMS shapefile\latest_network.shp
  • Field with section label.
  • Maximum distance: Only sections within this distance to points are considered. Decreasing this results in faster fitting.
  • Section length field: If set chainages are scaled so that point at end of section is this value.
  • Output layer.
